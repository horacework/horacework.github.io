试用markdown及其教程（网上抄的）

  • markdown

你是不是傻啊，写个文章都这么扯淡！~

想想还真是非常扯淡，从以前的写文章的编辑器，到现在的markdown语法，真是刷技能强行倒退体验。

到时候搞一个编辑器可以自动生成markdown格式文件就好了。TODOlist.add()

普及一波markdown语法吧

标题

标题前面加#号

一级标题：# XXXX
二级标题：## XXXX
三级标题：### XXXX
（如此类推）

一共可存在六级标题，#号后面须加空格！

列表

无序列表

文字前面加 - 或者 * 即可

  • 你好啊
  • 我好啊
  • 你是不是傻

有序列表

文字前面加 1. 2. 3. 即可

  1. 这是第一个
  2. 这是第二个
  3. 这是第三哥哥哥哥哥额

引用

文字前面加 >

点击左侧录制按钮即可开始录制本局比赛！

url链接

[baidu](http://baidu.com)

baidu

图片

相比链接，图片在前面多了个！

![img1](http://XXXXXXXX)

粗体与斜体

斜体：*斜体*
粗体：**粗体**

效果如下：斜体 ，粗体

代码框

使用两个`把中间代码包起来，func

分割线

分割线的语法只需要另起一行，连续输入三个星号 *** 即可。
